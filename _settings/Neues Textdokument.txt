Guide:
Extract the .zip file to main directory of StartIsBack (usually %LocalAppData%/StartIsBack).  
Set the taskbar theme to Classic3.  
For StartIsBack-only users: Set the Start orb to classic3small (Taskbar) or classic3big (Superbar).  
For StartIsBack (taskbar) + Classic Shell (Start menu) users: Set the Start orb in SIB to null_classic3small/big. Then set the Start button in Classic Shell settings to CS_classic3small/big.
